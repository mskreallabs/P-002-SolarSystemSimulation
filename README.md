# 🌌 Overview

Solar System Simulation is a Unity 3D project that recreates the planets of our Solar System using real-time orbital motion and self-rotation.

The project is designed for learning Unity fundamentals, astronomy visualization, and 3D simulation development.

---

# ✨ Features

## Planet Simulation

- Sun at the center
- Mercury
- Venus
- Earth
- Mars
- Jupiter
- Saturn
- Uranus
- Neptune

## Camera

- Click a planet to focus on it
- Reset camera to default target
- Adjustable camera distance
- Object tracking
- Smooth viewing experience

## Simulation

- Planet rotation
- Planet orbit
- Real-time movement
- Configurable speeds
- Independent planet settings

## Utility

- Screenshot system
- Prefab-based architecture
- Modular scripts
- Easy customization

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/MSKRealLabs/Solar-System-Simulation.git
```

Open the project using **Unity 6** (or Unity 2022 LTS if compatible).

Allow Unity to restore all packages automatically.

Press **Play**.
